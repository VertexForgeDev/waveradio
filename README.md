💿 WaveRadio

WaveRadio is a minimalist, high-energy web audio player and Progressive Web App (PWA) dedicated to pure electronic, dance, and trance music. Designed with a fully responsive Tailwind CSS dark interface, it maximizes web capabilities to deliver a seamless, immersive listening experience.

✨ Features

* **Curated & Custom Streams:** Pre-configured electronic stations with options to stream custom external URLs.
* **Local Playlist Support:** Seamlessly load and play local audio files directly in your browser.
* **Advanced Web Audio API:** Includes an interactive HTML5 canvas audio visualizer (multiple modes) and a 3-band parametric equalizer with electronic genre presets.
* **Nostalgic Atmosphere:** Features an optional vinyl record static/noise filter for a retro aesthetic.
* **Modern Integration:** Fully PWA-installable with service worker caching and integrated with the Media Session API for responsive lock-screen and background audio controls.
* **Utility Tools:** Built-in 30-minute sleep timer, automated dynamic browser titles, and a localized persistent history tracker.
